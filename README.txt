Analyzing the NBER patent database (Hall, Jaffe and Trajtenberg):

  <http://www.ckan.net/package/read/patent-nber>

## Files

Generally comments are in files but some exceptions:

* missing_patents.txt: list of patent numbers which are in the cite file but
  *not* in the patent file (and so needed to be added in when building the db).
  Originate from code in the convert_to_db.py file.
